# Node.js 100 Snippets (Repo Zip)

This repo contains **100 practical Node.js snippets** (ES Modules) organized as individual files.

## Requirements
- Node.js **20+**
- (Optional) install deps for framework/db/auth snippets: `npm install`

## Quick Start
```bash
npm install
node snippets/01-http-hello.js
```

## Organization
- `snippets/` — runnable snippet files
- `docs/INDEX.md` — list of all snippets with a short description and how to run them

## Notes
Some snippets require environment variables (e.g. `JWT_SECRET`, `DATABASE_URL`, `MONGO_URL`).  
See the per-snippet comments in the files.
